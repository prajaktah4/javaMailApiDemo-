Developed an email-sending application using Spring Boot and JavaMail API to automate email communication. The system allows sending plain text, HTML, and attachment-based emails through an SMTP server. 
It demonstrates seamless integration of Spring Boot with external mail services and efficient configuration management. 
The project focuses on dependency injection, Spring Boot configuration, and service-layer abstraction to ensure scalability and maintainability.
